# ChronicleX · 思源笔记日历事项插件

[English](https://github.com/wj1002/plugin-ChronicleX-vite-svelte/blob/main/README.md) | 简体中文

一个以**月历**为核心的**任务管理**插件:在日历上记录待办、切换状态、完成留痕,并能**一键关联思源文档**作为详情。

---

## ✨ 主要功能

### 📅 双入口视图
- **Dock 侧栏**:常驻右侧,包含迷你月历 + 当日清单(左键 mini cell 可跨日浏览)+ 过期事项可折叠分组
- **Tab 全屏月视图**:7×6 月格(42 cell),每日事项以**标题条**呈现(按优先级染色左边;每格最多 3 条 + "+N";已完成/取消变暗仍可见)
- **顶栏入口 + 快捷键**:点顶栏 📅 图标打开全屏 Tab;或按命令快捷键 `⇧⌘L` / `Shift+Ctrl+L`

### 🖐️ 拖拽与快捷操作
- **拖拽改日期**:月视图按住事项条拖到另一格即可改日期,桌面鼠标 + 移动触摸(长按)统一(重复事项实例不可拖,用编辑弹窗改期)
- **格子快捷新建**:每个日期格右上角 hover 出 `+`,一键新建当日事项
- **点条编辑 / 点空白看详情**:点事项条直接打开编辑;点格子空白处弹当日清单

### 📱 移动端支持
- **思源手机端可用**:桌面专属交互都补了触摸路径,在思源手机 App / 移动浏览器可用(v0.6.0 起)
- **长按菜单**:月视图日期格 / 迷你日历 cell / 状态按钮**长按**,弹出与桌面右键同款菜单(新建 / 复制 / 全状态切换)
- **左滑删除**:事项卡 iOS 风左滑露出删除(swipe-to-delete,滑回可取消)
- **底部操作栏 + 悬浮新建**:移动端底部固定栏(今天 / 搜索 / 筛选)+ 右下 FAB「＋」,搜索/状态/标签筛选下放到底部弹出面板(v0.6.3)
- **拖拽排序**:某天详情弹窗进「排序模式」,按住 ≡ 手柄拖动调整当天事项顺序并持久化(v0.6.4;重复事项沉底、不可拖)
- **窄屏弹窗适配**:编辑 / 文档选择 / 某天详情 / 范围选择等弹窗在窄屏手机自适应宽度、不横向溢出(v0.6.5)

### ✅ 事项管理
- **4 态状态**:待办 / 进行中 / 已完成 / 已取消
- **3 级优先级**:高 / 中 / 低,事项条左边按优先级染色
- **多标签**:自由打,自动聚合可筛选
- **状态切换**:左键 icon 一键切完成,右键弹全状态菜单
- **完成时自动记录** `completedAt`,便于后期复盘

### 🔗 思源文档关联(核心特色)
- **一键建文档**:首次选默认笔记本,之后自动用配置
- **新文档自动预填**:事项元信息卡(日期 / 状态 / 优先级 / 标签 / ID)
- **绑定已有文档**:全文搜索弹窗,250ms 防抖
- **解除绑定**:仅清除关联,不删除思源文档
- **打开关联文档**:点 📄 图标跳转(失效时 toast 提示)

### 🔁 重复事项
- **频率支持**:每天 / 每周 / 每月 / 每年
- **间隔设置**:每 N 天/周/月/年
- **过滤规则**:按周几(weekly)/ 按月日(monthly)
- **结束条件**:永不结束 / 重复 N 次 / 截止日期
- **范围修改**:单个实例 / 此日期及以后 / 整个序列

### 🎨 界面设计
- **EventCard**:优先级短条 + 标签 pill + 状态 4 态视觉
- **EventEditor**:分组卡片 + SegmentedControl
- **DayDetailDialog**:节假日 chip + 空态 CTA 按钮
- **DocPicker**:搜索输入 + GroupCard 列表
- **跟随思源主题**:所有视觉 token 使用 `var(--b3-*)` 变量,深浅色自动适配

### 🎉 中国法定节假日
- **数据源**:NateScarlet/holiday-cn
- **本地缓存**:按年存储到 plugin storage
- **自动更新**:可配「每月 / 每年 / 手动」,30 天阈值自动重拉
- **休/班 区分**:月视图格子日期旁显示「休」(放假,红)/「班」(调休上班,灰)角标 + 节日名;顶部「本月节假日」栏与 Dock chip 用 `休 …` / `班 …` 前缀,不再把调休标成放假
- **配色**:放假**淡红** / 调休**淡灰**,全主题 token 化,深浅色自适配
- **节假日倒计时**:月视图顶部栏右侧显示「距 {节日名} 还有 N 天」,当天正逢放假日时显示「{节日名} · 假期中」
- **UI 集成**:Dock MiniCalendar / 月视图 DayCell / 月视图横幅 / DayDetailDialog 等统一显示
- **可选开关**:整体开关 / 显示标记 / 显示节名

### 🔍 搜索与筛选
- **关键字搜索**:标题 + 备注模糊匹配(月视图工具栏)
- **状态筛选**:4 态 chips 控制事项条/点透明度 — 选中态 100% 显示,未选态变暗(不隐藏,仍可看出"那天有事项")
- **标签筛选**:自动从全部事项收集
- **当日详情浮窗**:左键 Tab 月视图格子**空白处** → 看清单 → 点事项编辑 / 点 `+` 新建
- **绑定文档搜索**(DocPicker):标题/全文搜索,250ms 防抖

### 🗑️ 删除功能
- **桌面**:事项卡 hover 浮现红色圆形 `−` 按钮
- **移动端**:事项卡 **iOS 风左滑**露出删除区(swipe-to-delete,滑回可取消)
- **删除重复事项**:弹 ScopeSelector 选择范围
- **删除关联文档事项**:只清事项,不影响思源文档

## 📥 安装

1. 到 [Releases](https://github.com/wj1002/plugin-ChronicleX-vite-svelte/releases) 下载最新 `package.zip`
2. 解压到 `{思源工作空间}/data/plugins/plugin-ChronicleX-vite-svelte/`
3. 重启思源,在「设置 → 集市 → 已下载」中启用

## 🚀 快速上手

> 📱 移动端:下面凡是「右键」改用「长按」,删除改用事项卡「左滑」。

启用后:

1. **右侧 Dock** 出现 📅 ChronicleX,打开它看到迷你月历 + 今日清单
2. **新建事项**:
   - Dock 顶部 `+` 按钮 → 今日快建
   - Dock 迷你日历 / Tab 月视图,**右键任意日子** → 菜单建事项
   - Tab 月视图**格子右上角 `+`** → 当日快建;或左键格子空白 → 当日详情浮窗 → 浮窗内 `+`
3. **改日期**:Tab 月视图直接**拖事项条**到目标格(重复事项用编辑弹窗改期)
4. **打开全屏视图**:点顶栏 📅 / 按 `⇧⌘L` / `Shift+Ctrl+L`
5. **关联文档**:在事项编辑器底部,点 `+ 一键建文档`(首次选笔记本)或 `⊕ 绑定已有文档`(搜索器)
6. **状态切换**:Dock 列表左键 `○` → `●`(标完成);右键 `○` → 4 态菜单
7. **筛选**:Tab 工具栏 chip 切换,搜索框模糊匹配标题/备注

## ⚙️ 设置项

「设置 → 集市 → 已下载 → ChronicleX → 齿轮」:

| 设置 | 默认 | 说明 |
|---|---|---|
| 默认笔记本 | (未选择) | 「一键建文档」生成的文档存放笔记本 |
| 默认文档路径 | `/ChronicleX` | 笔记本下的相对路径 |
| 一周从哪天开始 | 周一 | 月历的第一列 |
| 节假日显示 | 开启 | 是否显示中国法定节假日 |
| 节假日更新频率 | 每月 | 自动拉取节假日数据的频率 |
| 显示标记 / 显示节名 | 开启 | 月格上是否显示「休/班」角标及节日名称 |

- **v0.6.4**(2026-05-31):移动端 **拖拽排序**——某天详情弹窗「排序模式 + ≡ 手柄」拖动调整当天事项顺序并持久化(完全手动接管;重复事项不可拖、沉底);仅移动端,桌面零副作用。测试 214 → 233
- **v0.6.3**(2026-05-31):移动端 **底部操作栏 + 悬浮新建**——底部固定栏(今天 / 搜索 / 筛选)+ 右下 FAB「＋」,顶部搜索 / 状态 / 标签筛选下放到底部弹出面板;桌面零副作用。测试 200 → 214
- **v0.6.2**(2026-05-30):月视图顶部栏加**下一个节假日倒计时**——「距 端午节 还有 N 天」,当天放假时显示「端午节 · 假期中」;纯展示,桌面 / 移动通用。
- **v0.6.1**(2026-05-30):移动端 **swipe-to-delete**——事项卡 iOS 风左滑露出删除(点删 / 滑回取消,天然防误删),替代移动端常显的 `−` 按钮;桌面端零副作用(hover `−` 不变)。测试 177 → 189
- **v0.6.0**(2026-05-30):**移动端首次开启**——`frontends` 加 `mobile` / `browser-mobile`,插件正式在思源手机端可用;手机浏览器真机走查通过,核心链路(月历 / 今日清单 / 新建编辑 / 关联文档 / 节假日 / 长按菜单 / 拖拽改期)可达可用、无功能性阻断;设置面板窄屏宽度自适应。桌面端零回归。窄屏布局优化与移动专属交互留后续版本。
- **v0.5.2**(2026-05-30):移动端长按菜单适配(预埋)——为桌面右键菜单补一条触摸「长按」触发路径(月视图日期格 / Dock 迷你日历 / 事项卡状态按钮),按住 500ms 弹出与右键同款菜单。本版 `frontends` 仍未开启 mobile、作为预埋代码未经真机验证;**桌面端行为零改动**(右键 / 拖拽 / hover 全部不变)。测试 169 → 177
- **v0.5.1**(2026-05-29):节假日显示调整——放假/调休换配色(淡红/淡灰),月视图加「休/班」角标(节日名保留),调休不再标"(调休)"而用「班」前缀;月视图格子 / 顶部栏 / Dock 迷你日历三处统一。测试 166 → 169
