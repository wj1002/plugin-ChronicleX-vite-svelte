# ChronicleX · 思源笔记日历事项插件

[English](https://github.com/wj1002/plugin-ChronicleX-vite-svelte/blob/main/README.md) | 简体中文

一个以**月历**为核心的**任务管理**插件:在日历上记录待办、切换状态、完成留痕,并能**一键关联思源文档**作为详情。

## ✨ 主要功能

### 📅 双入口视图
- **Dock 侧栏**:常驻右侧,包含迷你月历 + 当日清单(左键 mini cell 可跨日浏览,**双击日期打开/创建思源每日笔记**)+ 过期事项可折叠分组
- **Tab 全屏月视图**:7×6 月格(42 cell),每日事项以**标题条**呈现(按优先级染色左边;每格最多 5 条 + "+N";已完成/取消变暗仍可见)
- **顶栏入口 + 快捷键**:点顶栏 📅 图标打开全屏 Tab;或按命令快捷键 `⇧⌘L` / `Shift+Ctrl+L`

### 🖐️ 拖拽与快捷操作
- **拖拽改日期**:月视图按住事项条拖到另一格即可改日期,桌面鼠标 + 移动触摸(长按)统一(重复事项实例不可拖,用编辑弹窗改期)
- **格子快捷新建**:每个日期格右上角 hover 出 `+`,一键新建当日事项
- **点条编辑 / 点空白看详情**:点事项条直接打开编辑;点格子空白处弹当日清单

### 📱 移动端支持
- **思源手机端可用**:桌面专属交互都补了触摸路径,在思源手机 App / 移动浏览器可用(v0.6.0 起)
- **长按菜单**:月视图日期格 / 迷你日历 cell / 状态按钮**长按**,弹出与桌面右键同款菜单(新建 / 复制 / 全状态切换)
- **左滑删除**:事项卡 iOS 风左滑露出删除(swipe-to-delete,滑回可取消)
- **底部操作栏 + 悬浮新建**:移动端底部固定栏(今天 / 搜索 / 筛选)+ 右下 FAB「＋」,搜索/状态/标签筛选下放到底部弹出面板(v0.6.3)
- **拖拽排序**:某天详情弹窗进「排序模式」,按住 ≡ 手柄拖动调整当天事项顺序并持久化(v0.6.4;重复事项沉底、不可拖)
- **窄屏弹窗适配**:编辑 / 文档选择 / 某天详情 / 范围选择等弹窗在窄屏手机自适应宽度、不横向溢出(v0.6.5)
- **窄屏布局优化**:手机端月视图格子更紧凑(更小内距、更矮格高、节日名让位给「休/班」角标),设置面板行纵向堆叠、控件占满宽,编辑弹窗时间段整体掉到独立满宽行(v0.6.6)

### ✅ 事项管理
- **4 态状态**:待办 / 进行中 / 已完成 / 已取消
- **3 级优先级**:高 / 中 / 低,事项条左边按优先级染色
- **多标签**:自由打,自动聚合可筛选
- **状态切换**:左键 icon 一键切完成,右键弹全状态菜单
- **完成时自动记录** `completedAt`,便于后期复盘

### 🔗 思源文档关联(核心特色)
- **一键建文档**:首次选默认笔记本,之后自动用配置;文档路径支持模板变量按日期自动归档
- **默认创建文档(v1.10.0,可选,默认关)**:开启设置项「默认创建文档」后,保存新建事项自动建并绑文档;未配默认笔记本时静默 toast 跳过、不打断保存;编辑已有事项不触发
- **新文档自动预填**:事项元信息卡(日期 / 状态 / 优先级 / 标签 / ID)
- **绑定已有文档**:全文搜索弹窗,250ms 防抖
- **解除绑定**:仅清除关联,不删除思源文档
- **打开关联文档**:点 📄 图标跳转(失效时 toast 提示)
- **反查关联事项(v0.7.0,v0.7.2 修正触发位置)**:在思源**文档树右键菜单**点「ChronicleX 关联事项 (N)」,列出关联本文档的事项;每行可「编辑」或「跳日历」定位到事项日期
- **双击开每日笔记(v1.10.0)**:双击 Dock 迷你日历日期开 / 建思源**每日笔记**——今天幂等建,历史日 SQL 反查已有(无则提示、不补建);无笔记本配每日笔记路径时提示去配置。设置项「每日笔记笔记本」可指定使用哪个笔记本(留空 = 首个已配置)

### 🔁 重复事项
- **频率支持**:每天 / 每周 / 每月 / 每年
- **间隔设置**:每 N 天/周/月/年
- **过滤规则**:按周几(weekly)/ 按月日(monthly)
- **结束条件**:永不结束 / 重复 N 次 / 截止日期
- **范围修改**:单个实例 / 此日期及以后 / 整个序列

### 🎨 界面设计
- **Liquid Glass 液态玻璃(v1.11.0)**:全局液态玻璃设计语言——半透明磨砂卡 + 彩色 mesh 渐变背景折射 + 顶部高光 + 动态光效,统一 16px 圆角,适配亮 / 暗主题
- **液态玻璃 / 经典风格切换(v1.11.0)**:设置项「液态玻璃效果」一键开关,关闭回退经典不透明风格(Dock 等滚动容器已关 `backdrop-filter` 防卡顿)
- **时间线回放(v1.11.0)**:复盘页「回放」tab,选时间段逐天回放事项完成情况,滑块 + 播放 / 暂停 / 步进,自动跳过无事项天
- **EventCard**:优先级短条 + 标签 pill + 状态 4 态视觉
- **EventEditor**:分组卡片 + SegmentedControl
- **DayDetailDialog**:节假日 chip + 空态 CTA 按钮
- **DocPicker**:搜索输入 + GroupCard 列表
- **跟随思源主题**:所有视觉 token 使用 `var(--b3-*)` 变量,深浅色自动适配

### 🎉 中国法定节假日
- **数据源**:NateScarlet/holiday-cn
- **本地缓存**:按年存储到 plugin storage
- **自动更新**:可配「每月 / 每年 / 手动」,30 天阈值自动重拉
- **休/班 区分**:月视图格子日期旁显示「休」(放假,红)/「班」(调休上班,灰)角标 + 节日名;顶部「本月节假日」栏与 Dock chip 用 `休 …` / `班 …` 前缀,不再把调休标成放假
- **配色**:放假**淡红** / 调休**淡灰**,全主题 token 化,深浅色自适配
- **节假日倒计时**:月视图顶部栏右侧显示「距 {节日名} 还有 N 天」,当天正逢放假日时显示「{节日名} · 假期中」
- **UI 集成**:Dock MiniCalendar / 月视图 DayCell / 月视图横幅 / DayDetailDialog 等统一显示
- **可选开关**:整体开关 / 显示标记 / 显示节名

### 🔍 搜索与筛选
- **关键字搜索**:标题 + 备注模糊匹配(月视图工具栏)
- **状态筛选**:4 态 chips 控制事项条/点透明度 — 选中态 100% 显示,未选态变暗(不隐藏,仍可看出"那天有事项")
- **标签筛选**:自动从全部事项收集
- **当日详情浮窗**:左键 Tab 月视图格子**空白处** → 看清单 → 点事项编辑 / 点 `+` 新建
- **绑定文档搜索**(DocPicker):标题/全文搜索,250ms 防抖

### 🗑️ 删除功能
- **桌面**:事项卡 hover 浮现红色圆形 `−` 按钮
- **移动端**:事项卡 **iOS 风左滑**露出删除区(swipe-to-delete,滑回可取消)
- **删除重复事项**:弹 ScopeSelector 选择范围
- **删除关联文档事项**:只清事项,不影响思源文档

## 📥 安装

1. 到 [Releases](https://github.com/wj1002/plugin-ChronicleX-vite-svelte/releases) 下载最新 `package.zip`
2. 解压到 `{思源工作空间}/data/plugins/plugin-ChronicleX-vite-svelte/`
3. 重启思源,在「设置 → 集市 → 已下载」中启用

## 🚀 快速上手

> 📱 移动端:下面凡是「右键」改用「长按」,删除改用事项卡「左滑」。

启用后:

1. **右侧 Dock** 出现 📅 ChronicleX,打开它看到迷你月历 + 今日清单
2. **新建事项**:
   - Dock 顶部 `+` 按钮 → 今日快建
   - Dock 迷你日历 / Tab 月视图,**右键任意日子** → 菜单建事项
   - Tab 月视图**格子右上角 `+`** → 当日快建;或左键格子空白 → 当日详情浮窗 → 浮窗内 `+`
3. **改日期**:Tab 月视图直接**拖事项条**到目标格(重复事项用编辑弹窗改期)
4. **打开全屏视图**:点顶栏 📅 / 按 `⇧⌘L` / `Shift+Ctrl+L`
5. **关联文档**:在事项编辑器底部,点 `+ 一键建文档`(首次选笔记本)或 `⊕ 绑定已有文档`(搜索器)
6. **状态切换**:Dock 列表左键 `○` → `●`(标完成);右键 `○` → 4 态菜单
7. **筛选**:Tab 工具栏 chip 切换,搜索框模糊匹配标题/备注

## ⚙️ 设置项

「设置 → 集市 → 已下载 → ChronicleX → 齿轮」:

| 设置 | 默认 | 说明 |
|---|---|---|
| 默认笔记本 | (未选择) | 「一键建文档」生成的文档存放笔记本 |
| 默认文档路径 | `/ChronicleX` | 笔记本下的相对路径,支持模板变量: `{{date}}` `{{year}}` `{{month}}` `{{day}}` `{{title}}` 及 `{{now | date "YYYY-MM-DD"}}` 格式化 |
| 一周从哪天开始 | 周一 | 月历的第一列 |
| 优先级颜色 | 红/蓝/绿 | 高/普通/低 三级各选预设色,事项条与编辑器实时跟随 |
| 启用提醒 | 开启 | 事项到点弹系统桌面通知的总开关 |
| 全天事项提醒时刻 | 09:00 | 无具体时间的事项,按此时刻作提醒锚点 |
| 节假日显示 | 开启 | 是否显示中国法定节假日 |
| 节假日更新频率 | 每月 | 自动拉取节假日数据的频率 |
| 显示标记 / 显示节名 | 开启 | 月格上是否显示「休/班」角标及节日名称 |

## 🗺️ 版本历史 & 路线图

### 已发布

- **v1.12.0**(2026-06-22):**自动归档**——已完成 / 已取消事项超过 N 天自动从日常视图隐藏(统计数据保留),设置面板新增「自动归档」开关(默认开)+「归档天数」(1-365,默认 30)。插件启动及配置变更时自动执行归档扫描;设置面板「数据」段新增「管理已归档」弹窗(查看 / 恢复)。详见 [CHANGELOG](CHANGELOG.md)。测试 671
- **v1.11.1**(2026-06-18):**修复设置面板优先级色块重叠拥挤**——「颜色」段高 / 普通 / 低三级的预设色块此前用 `padding + 负 margin` 撑触摸目标,后补的 `padding: 0` 留下残余负 margin 把相邻色块互拉、吃掉 flex 间距致重叠拥挤。删负 margin + `gap` 8→10px + 补 `box-sizing` 修复。详见 [CHANGELOG](CHANGELOG.md)。测试 650
- **v1.11.0**(2026-06-16):**Liquid Glass 液态玻璃设计 + 时间线回放**——全局引入液态玻璃设计语言(半透明磨砂卡 + 彩色 mesh 折射 + 高光 / 动态光效 + 统一 16px 圆角,亮 / 暗主题自适应),主面板 / 复盘 / 纪念日 / 弹窗 / Dock 全线接入;设置项「液态玻璃效果」可**一键切换液态玻璃 / 经典风格**;复盘页新增「**时间线回放**」tab(选时间段逐天回放,滑块 + 播放 / 暂停 / 步进)。另:统一 frosted field 表单控件、自定义 ScrollSelect 替换原生长下拉、弹层 portal 逃逸玻璃容器;**修复**弹窗顶角白三角、Dock 滚动卡顿(关 `backdrop-filter`)、看板列内滚动 + 卡片防压扁、各开关变形。详见 [CHANGELOG](CHANGELOG.md)。测试 → 649
- **v1.10.1**(2026-06-12):**修复月历不显示重复事项**——周视图和日视图均调用 `expandEventsInRange` 展开重复事项实例,但月历直接读原始列表,导致重复事项在月历上不可见。v0.8.0 已知不一致,现统一修复。详见 [CHANGELOG](CHANGELOG.md)。测试 636
- **v1.10.0**(2026-06-12):**默认创建文档开关 + 双击侧边栏日期开每日笔记**——设置项「默认创建文档」开启后,保存新建事项自动建并绑思源文档(未配默认笔记本时静默跳过、不打断保存;编辑已有不触发);双击 Dock 迷你日历日期打开 / 创建思源每日笔记(今天幂等建,历史日 SQL 反查已有、无则提示不补建,无配置时提示去思源配置),单击仍切当日列表。每日笔记 SQL 反查经 box + date 双正则校验防注入。本版另补:文档路径模板新增 `YY`(两位年)token 支持 `YYMMDD` 等紧凑格式;侧边栏迷你日历底部列表新增二十四节气(受「显示农历」开关控制)。详见 [CHANGELOG](CHANGELOG.md)。测试 611 → 636
- **v1.9.0**(2026-06-11):**事项智能排序 + 完成庆祝动画**——今日清单可按截止日紧迫度 + 优先级 + 事项年龄自动排序(设置开关,默认关);标记完成时播放粒子爆炸动画(跟随优先级配色,`prefers-reduced-motion` 时跳过,默认开)。**修复**:「默认文档路径」保存时漏 `await` 导致写入 Promise 对象,引发 `[object Object]` 显示与 `t.replace is not a function` 建文档失败;`normalizeConfig` 补 `defaultDocPath` / `defaultNotebookId` / `weekStart` 类型守卫;`resolveDocPath` 反斜杠 `\` 自动转 `/` + 补前导 `/`(Windows 习惯路径如 `\2026\` 可正常建文档)。详见 [CHANGELOG](CHANGELOG.md)。测试 555 → 605
- **v1.8.0**(2026-06-10):**时间分布占比环形卡**——复盘页概览区新增「时间分布」环形卡(conic-gradient),按数量占比展示事项构成,单环 + tab 切换状态 / 优先级 / 标签三维度,跟随顶部周 / 月 / 年 / 全部选择器。状态、优先级显全部固定段(含 0 占比),标签按出现次数 Top 8 + 「其他」聚合、只显非零段;环中心显总数 + 单位(件 / 次),图例每段显数量 + 百分比。与现有「任务分布」横向条形形成「占比 vs 绝对量」互补。详见 [CHANGELOG](CHANGELOG.md)。测试 526 → 540
- **v1.7.1**(2026-06-09):**修复——同日法定假日与纪念节日并存显示**——此前一天同时命中法定节假日(如端午节调休放假日)和纪念节日(如父亲节,6 月第 3 个周日)时,法定优先、纪念被丢弃,导致 6-21 只显示「夏至 + 端午节」而看不到父亲节,与顶部「本月节假日」栏不一致。现两者并存:法定保留「休/班」角标 + 背景色 + 节日名,纪念作为独立一行显示。月视图、周视图同步生效。详见 [CHANGELOG](CHANGELOG.md)。测试 511 → 526
